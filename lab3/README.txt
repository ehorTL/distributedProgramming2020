Программа демонстрирует общение клиента с сервером по протоколу TCP/IP с использованем сокетов.
В демонстративных целях оба приложения клиента и сервера запускаются параллельно в разных потоках.
Клиент выполняет скрипт обращения к серверу с запросом всех Стран, а затем всех Городов, имеющихся в базе данных.
Сервер, обращается к базе данных с помощью JDBC.
После запуска сервер ожидает подключения клиента и переходит в режим общения, то есть ожидания запросов от клиента, а затем ответа на них.
Клиент после запуска подсоединяется к серверу, выполняет пользовательский скрипт и завершает работу.
Для отправки данных используются сериализированные объекты.
