Данная программа, демонстрирует сценарий взаимодействия с базой данных посредством прямого обращения к ней (через JDBC).
Для демонстративныз целей была выбрана удаленная БД MySQL.
Скрипт создания базы данных находится в файле tables.sql.
Информация с паролями к БД в файле DB_info.txt.
Для выполнения сценария нужно пошагово выполнить инструкции в файле TEST_CASE.txt,
Выполняя ТОЛЬКО ОДНУ инструкцию за один запуск программы и сверяя результаты выведенной информации До и После изменений.

Исключения не обрабатываются.
Могут возникать изза плохого интернет соединения.
